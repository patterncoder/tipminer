# menuItem

Represents any item (usually food or beverage) for sale by a restaurant.


| Fields        | Type    | Description
| ------------- | ------- | ------------|
| name          | String  | Name of the item |
| Description   | String  | Description of the item |
| Category      | String  | Type of food/beverage |
